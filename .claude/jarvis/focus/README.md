# jarvis 집중 영역 (focus)

이 디렉토리에 **이 프로젝트에서 특히 주의 깊게 리뷰받고 싶은 내용**을 마크다운으로 모아둔다.
`jarvis-once`가 리뷰할 때 여기 문서들을 **"집중 영역" 맥락으로, 일반 감지 카탈로그보다 우선**해서 본다.

두루뭉술한 보편 컨텍스트 대신 **구체적이고 프로젝트 고유한 주의점**을 적을수록 navigation이 정확해진다.

## 무엇을 넣나 (예)
- `concurrency.md` — 이 서비스의 동시성/락 규칙, 건드리면 위험한 공유 상태
- `payment-invariants.md` — 결제 금액 계산에서 절대 깨지면 안 되는 불변식
- `known-traps.md` — 과거에 사고 났던 지점, 반복되는 실수 패턴

파일은 자유 형식이다. 이 README만 있고 다른 문서가 없으면 집중 영역 없이 기본 리뷰만 한다.

> 이 디렉토리는 커밋되어 **팀과 공유**될 수 있다. 개인용으로만 쓰려면 `.gitignore`에 추가한다.
> 위치를 바꾸려면 `/jarvis focus=<경로>` 인자를 쓴다(기본 `.claude/jarvis/focus/`).
